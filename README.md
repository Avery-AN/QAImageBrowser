# QAImageBrowser
图片浏览器<br>
=========

主要功能点:<br>
--------------
1. 支持对image or url的全屏显示 <br>
2. 支持单击、双击、缩放以及拖动退出等手势 <br>
3. 支持gif <br>

主要特色:<br>
--------------
gif的展示从缩略图到全屏然后再回到缩略图的整个过程、gif不断片、不掉帧。
